HƯỚNG DẪN TẠO FONT UNITY CHO BEPINEX

Tài liệu chi tiết từng bước đóng gói Font (TextMeshPro) thành AssetBundle dùng cho Mod/Việt Hóa game Unity.

1 Chuẩn Bị Công Cụ

  • Unity Hub & Unity Editor: Bạn cần tải Unity Hub và cài đặt phiên bản Unity Editor cùng phiên bản (hoặc gần giống nhất) với tựa game bạn định mod. (Mẹo: Chuột phải vào file .exe của game -> Properties -> Details để xem version, hoặc dùng tool như UnityCrashHandler).
  • File Font chữ: Một file font hỗ trợ tiếng Việt định dạng .ttf hoặc .otf (VD: Arial, Roboto, BeVietnamPro...).
  • Môi trường code Mod: Đã setup sẵn project C# BepInEx.

2 Tạo Project & Import Font

1. Mở Unity Hub, tạo một Project mới (chọn mẫu 3D Core hoặc 2D Core đều được).

2. Chờ Unity load xong, tại cửa sổ Project (thường ở góc dưới), tạo một thư mục tên là MyFonts.

3. Kéo thả file .ttf hoặc .otf của bạn vào thư mục MyFonts vừa tạo trong Unity.

3 Tạo TextMeshPro Font Asset

Hầu hết các game Unity hiện đại đều dùng TextMeshPro (TMP) thay vì UI Text cũ. Do đó, ta cần tạo TMP Font Asset.

1. Trên thanh menu trên cùng, chọn Window > TextMeshPro > Font Asset Creator.

2. Cửa sổ Font Asset Creator hiện ra, bạn cần thiết lập như sau:

  • Source Font File: Kéo file font của bạn từ dưới thẻ Project thả vào ô này.
  • Sampling Point Size: Auto
  • Padding: 5 (hoặc 7, 9 tùy font).
  • Atlas Resolution: 4096 x 4096 (Rất quan trọng! Để size lớn mới đủ chỗ chứa hết ký tự tiếng Việt, nếu nhỏ quá chữ sẽ bị mất).
  • Character Set: Chọn Custom Range.
  • Character Sequence (Mã Unicode Tiếng Việt): Copy và dán chính xác dòng sau vào ô này:
32-126, 160-255, 258-259, 272-273, 296-297, 360-361, 416-417, 431-432, 7840-7929

3. Nhấn nút Generate Font Atlas. Đợi vài giây để Unity render ảnh.

4. Sau khi render xong (kiểm tra xem có ký tự nào bị Missing không, nếu missing nhiều tức là file font gốc không hỗ trợ tiếng Việt hoặc Atlas nhỏ quá), nhấn Save và lưu file với tên (ví dụ: TiengVietFont.asset) vào thư mục MyFonts.

4 Đóng Gói Thành AssetBundle

BepInEx không thể đọc trực tiếp file .asset của Editor, mà phải thông qua AssetBundle.

1. Chọn file TiengVietFont.asset vừa tạo ra.

2. Nhìn sang cột Inspector (bên phải màn hình), cuộn xuống tận cùng dưới cùng.

3. Tìm chỗ có chữ AssetBundle. Nhấp vào chữ None -> chọn New... và gõ tên bundle (viết liền không dấu, ví dụ: vietnamese_font). Cột bên cạnh (Variant) để trống.

Lưu ý: Unity mặc định không có nút "Bấm để tạo Bundle". Bạn phải dùng một đoạn script nhỏ để thêm nút này.

4. Trong cửa sổ Project, tạo một thư mục tên là Editor (viết hoa chữ E).

5. Click chuột phải vào thư mục Editor > Create > C# Script (hoặc MonoBehaviour Script tùy phiên bản). Đặt tên là BuildAssetBundle.

6. Mở file đó lên và dán đoạn code sau vào, sau đó Save lại:

using UnityEditor;
using System.IO;

public class BuildAssetBundle
{
    [MenuItem("Assets/Build AssetBundles")]
    static void BuildAllAssetBundles()
    {
        string assetBundleDirectory = "Assets/AssetBundles";
        if (!Directory.Exists(assetBundleDirectory))
        {
            Directory.CreateDirectory(assetBundleDirectory);
        }
        
        /* Target Windows 64-bit (chỉnh sửa nếu game chạy HĐH khác) */
        BuildPipeline.BuildAssetBundles(assetBundleDirectory, 
                                        BuildAssetBundleOptions.None, 
                                        BuildTarget.StandaloneWindows64);
    }
}

7. Trở lại Unity, đợi script compile xong. Nhìn lên thanh Menu trên cùng, bạn sẽ thấy menu Assets > Build AssetBundles. Nhấn vào đó!

⚠️ Xử lý lỗi: Nếu Menu "Build AssetBundles" không xuất hiện?

  • Chưa xóa trắng code cũ: Khi tạo file C# mới, Unity tự sinh ra code kế thừa MonoBehaviour. Bạn hãy xóa sạch 100% code trong file đó và dán lại đoạn code trên.
  • Lỗi biên dịch (Console Error): Mở thẻ Console (Window > General > Console). Nếu project của bạn đang có bất kỳ lỗi màu đỏ nào (dù ở file khác), Unity sẽ ngưng cập nhật UI/Menu. Hãy clear lỗi đó.
  • Nhấn Refresh: Bấm chuột vào màn hình Unity và nhấn Ctrl + R để ép phần mềm quét lại code.

8. Đợi chạy xong, vào thư mục project của bạn (Mở File Explorer), vào Assets/AssetBundles. File có tên vietnamese_font (không có đuôi .manifest) chính là file bạn cần copy vào mod.

5 Cách Load Font Bằng BepInEx Plugin (C#)

Bây giờ bạn copy file vietnamese_font bỏ vào thư mục plugin của bạn. Dưới đây là code mẫu để load font và gán cho game:

using BepInEx;
using UnityEngine;
using TMPro; /* Nhớ tham chiếu (reference) đến Unity.TextMeshPro.dll */

[BepInPlugin("com.yourname.modname", "Viet Hoa Font Mod", "1.0.0")]
public class Plugin : BaseUnityPlugin
{
    public static TMP_FontAsset vnFont;

    private void Awake()
    {
        /* 1. Load file AssetBundle từ đường dẫn */
        /* Lấy thư mục chứa file dll của plugin hiện tại */
        string bundlePath = System.IO.Path.Combine(Paths.PluginPath, "TênThưMụcModCủaBạn", "vietnamese_font");
        
        AssetBundle bundle = AssetBundle.LoadFromFile(bundlePath);
        
        if (bundle == null)
        {
            Logger.LogError("Không thể load AssetBundle!");
            return;
        }

        /* 2. Load TMP_FontAsset từ Bundle ra bộ nhớ */
        /* Tên "TiengVietFont" là tên của file .asset bạn đã lưu ở Bước 3 */
        vnFont = bundle.LoadAsset<TMP_FontAsset>("TiengVietFont");

        if (vnFont == null)
        {
            Logger.LogError("Không tìm thấy font trong Bundle!");
        }
        else
        {
            Logger.LogInfo("Load Font Thành Công!");
            /* Giữ cho font không bị xóa khi chuyển cảnh */
            DontDestroyOnLoad(vnFont); 
        }

        /* Đừng quên giải phóng bộ nhớ Bundle sau khi lấy xong font (chỉ Unload bundle, giữ lại asset) */
        bundle.Unload(false);
    }

    /* Ví dụ cách áp dụng font vào game bằng phím tắt */
    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.F2))
        {
            ApplyFontToAllTexts();
        }
    }

    private void ApplyFontToAllTexts()
    {
        if (vnFont == null) return;

        /* Tìm tất cả các TextMeshPro UI trong game hiện tại */
        TextMeshProUGUI[] allTexts = Resources.FindObjectsOfTypeAll<TextMeshProUGUI>();
        foreach (var text in allTexts)
        {
            /* 1. Lưu lại shader gốc của game đang dùng */
            Shader originalShader = null;
            if (text.fontSharedMaterial != null)
            {
                originalShader = text.fontSharedMaterial.shader;
            }

            /* 2. Đổi font và material */
            text.font = vnFont;
            text.fontSharedMaterial = vnFont.material; 

            /* 3. Phủ lại shader gốc lên font mới để xóa lớp sương mờ */
            if (originalShader != null && text.fontSharedMaterial != null)
            {
                text.fontSharedMaterial.shader = originalShader;
            }
        }
    }
}

6 Các Lỗi Thường Gặp (Troubleshooting)

⚠️ Chữ bị viền sương mờ đục, các ô vuông mờ bao quanh?

Đây là lỗi Mất Shader SDF. Đã được fix trong đoạn code C# ở bước 5. Nguyên tắc là phải lấy shader từ fontSharedMaterial cũ của game gán đè lên material của font mới.

ℹ️ Hiện ô vuông (Missing Character) dù đã đổi font?

Do ở Bước 3 bạn quên nhập đoạn mã dải Unicode Tiếng Việt, hoặc size Atlas quá nhỏ (phải để 4096x4096) khiến font bị tràn và không chứa được các ký tự có dấu.

Chúc bạn thành công với dự án Mod / Việt Hóa của mình!

Ghi chú: Nếu game dùng Font Text cơ bản (không phải TextMeshPro), ở bước 4 bạn chỉ việc gán AssetBundle trực tiếp vào file .ttf gốc và LoadAsset dạng <Font> thay vì <TMP_FontAsset>.
